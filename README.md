# house-rent-app
A full-stack web application that allows users to register, log in, browse available rental properties, add new property listings, and book rental homes online. The system is built using React.js for the frontend and Node.js with Express.js for the backend, demonstrating REST API integration and full-stack development concepts.
# House Rent Application (MERN Stack)

This project is a full-stack web application developed using the MERN stack (MongoDB, Express.js, React.js, Node.js). The application allows users to register, log in, view available rental properties, add new properties, and book rental homes.

## Features
- User Registration and Login
- Add Property Listings
- View Available Properties with Images
- Book Rental Properties
- REST API Integration between Frontend and Backend

## Technologies Used
Frontend: React.js, Bootstrap, Axios  
Backend: Node.js, Express.js  
Database: MongoDB (or temporary storage for demo)

## Project Structure
- frontend: React UI
- backend: Express API server
